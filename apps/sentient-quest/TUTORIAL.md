# 🎮 SentientQuest: Quick Start Tutorial

**Last Modified:** 2026-04-26T17:44:55Z
**Timestamp:** 2026-04-26T17:44:55Z

Welcome, Architect. This tutorial will guide you through your first Genesis sequence.

## 1. The Core Loop

1. **Genesis**: Define your school's nature, theme, and locus.
2. **Simulation**: Watch as spirits and entities populate the graph.
3. **Observation**: Use the Telemetry Stream to monitor systemic changes.
4. **Intervention**: Click nodes to focus on them and influence their state.

## 2. Advanced Features

### 🔊 Kokoro TTS (Text-to-Speech)

To hear the voices of the spirits:

1. Ensure a **Kokoro TTS Server** is running (default: `http://localhost:8888`).
2. The game will automatically send narration and dialogue to the server.
3. You can change the voice in the Settings panel.

### 🤖 Autoplay Mode

Want to watch the world evolve hands-free?

1. Open the **System Monitor**.
2. Toggle **Autoplay: Enabled**.
3. The simulation will periodically trigger reality shifts and progress the narrative automatically.

## 3. Keyboard Shortcuts

- **[Space]**: Toggle Simulation Pause/Resume.
- **[R]**: Trigger Reality Reset (Careful!).
- **[D]**: Toggle Developer Mode overlays.

## 4. Troubleshooting

- **No Sound?** Verify your Kokoro server is reachable via browser.
- **Lag?** Reduce the `textConcurrency` in Settings.
